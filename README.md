# DY5WORD
Version 2 of Dynword (Wordle for Revit)

Make sure your Revit file is totally blank - no hidden filled regions! 

Click run and pick 4, 5, 6, or 7 letters from the drop down. 

Zoom to fit so you can see the boxes and the keyboard.

Click run and enter your word.

Repeat until the game is over.

Hit run again and select the red button (New Game).

Enjoy!
